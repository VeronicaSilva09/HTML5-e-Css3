Código-fonte / Pré-formatação

O comando document.getElementById('teste) é escrito em linguagem JavaScript. Ao usar a tag code no código js ele pré-formatou a fonte para deixar em formato de código monospace.


Abaixo temos um exemplo de um código escrito em Python, utilizando a tag pre juntamente com code para deixar o texto dentro dela exatamente igual e formatada.

        
num= int(input('Digite um número'))
if num % 2 == 0:
        print(f'O número {num} é PAR')
else:
        print(f'O número {num} é IMPAR')
print('Fim do programa')
