Entendendo a Hierarquia de Títulos em HTML

1. Introdução

Utilização das tags de titulos, a tag <h1> é utilizada para o título principal da página (Entendendo a Hierarquia de Títulos em HTML).

Para organizar uma página hierarquicamente após a tag <h1> consequentemente será utilizada a tag <h2> (Introdução). As outras tags serão explicadas conforme cada título.

1.1 tag <h3>

A tag <h3> serve como se fosse um nivel de sub-título para a tag <h2>.

1.1.1 tag <h4>

A tag <h4> é utilizado para um nivel de sub-título da tag <h3> e existem outros niveis hierarquicos, que vão até o <h6> que deverão ser ultilizados da forma correta e respectiva.

2. Lorem

O lorem é um texto automatico utilizado para testar a semántica da página HTML, antes de inserir o texto da páginas reais. No título 2.1 será utilizado um exemplo de lorem.
