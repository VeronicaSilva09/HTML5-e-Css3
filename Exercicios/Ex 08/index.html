Principais formatações

Negrito / Destaque

Negrito sem semântica utilizando a tag <b>, é apenas um formato sem significado, uma forma.

Nesta frase temos um destaque utilizando a tag <strong> que dará semântica a palavra ou frase destacada dentro dela.

Itálico / Ênfase

Nesta frase,temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em>(semântica)

Marcação

Podemos criar texto marcado utilizando a tag <mark>. Este é um exeplo de texto marcado com semântica, para dar sentido a frase, quaisquer outro tipo de marcação pode ser feito em CSS.
Para alterar a cor do Mark, podemos usar a <style> dentro da tag mark,a tag final para alterar a cor ficaria dessa forma: <mark <style background-color: blue;>> </mark>

Para deixar salvo que todos os mark fique com a cor de que você deseja é só abrir uma tag style no head e pronto.

Texto Deletado

Podemos marca um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marca um texto como inserido para dar uma ênfase e indicar que eke foi adicionado depois.
