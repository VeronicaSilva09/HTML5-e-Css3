Lista Ordenada <ol>

  1. Trabalho
  2. Almoço
  3. Estudo
  4. Jantar

Se utiliza a tag <li> dentro da ol , para identificar cada item da lista. Ex:

        
<ol>
    <li>Trabalho</li>
    <li>Almoço</li>
    <li>Estudo</li>
    <li>Jantar</li>
</ol>

Exemplo de uso da tag <ol type="I" start="5">

  1. Banana
  2. Goiaba
  3. Maçã
  4. Uva
  5. Morango

Neste exemplo acima, como diz o título, foi utilizado dois parâmetros dentro da tag <ol>, o type indica o tipo de ordem que será utilizado, neste caso foi em algarismo romano, já o start indica a partir de qual número a ordem será iniciada, neste caso foi a partir da quinta casa do algarismo romano.

Lista de Definição

É como se fosse um dicionário, temos os termos e as suas descrições. É uma lista sem demarcadores, mas bem útil em alguns casos.

Toda lista de definição está dentro de uma tag <dl> </dl> (definition list), cada termo é um <dt> (definition term) e cada descrição é um <dd> (definition description). Veja o resultado do código abaixo:

            
 <dl>
    <dt>HTML</dt>
    <dd>Linguagem de marcação utilizada para criar o conteúdo de sites.</dd>

    <dt>CSS</dt>
    <dd>Linguagem de marcação para a especificação de estilos de sites</dd>

    <dt>JavaScript</dt>
    <dd>Linguagem de programação para criar interatividades em sites.</dd>
</dl>

Resultado:

HTML
Linguagem de marcação utilizada para criar o conteúdo de sites.
CSS
Linguagem de marcação para a especificação de estilos de sites
JavaScript
Linguagem de programação para criar interatividades em sites.

O siginificado desse tipo de listagem é relacionado diretamente aos pares Termo + Descrição e isso nos ajuda bastante com o mecanismo de busca.
